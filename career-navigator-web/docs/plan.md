## 1. รฅยฐรฆยกรฆยฆรฅยฟยต Project Concept

### รคยธรฅยฅรจยฉยฑรฅยฎรคยฝ
> **AI Career Navigator** รฆยฏรคยธรฅรงยตรฅ LLMรฃรฅยฑยฅรฆยญยทรฅรฆรฃรจยทรงยผยบรฉรฅยฐรจรฆยฑรจยทรจยฟยฝรจยนยครงรจยทรฆยถยฏรงยฎยกรงรฅยนยณรฅยฐรฃ
> รคยธรฅยณรฅยฑยฅรฆยญยท รข AI รฆยฝรฅยบรฆรจยฝ รข รจยฒยผรคยธรจยทรงยผยบรฆรจยฟยฐ รข รฅยพรฅยฐรฉรฅยฐรฅรงยดรจรงยผยบรฅยฃรฅรฆ รข รจยฟยฝรจยนยครฆรฉรฉยฒรฅยบยฆรฃ

### รฅยฐรฅยครจยชยชรฆยณรฏยผรฉยขรจยฉยฆรงยจรฏยผ
> "AI Career Navigator is a career management platform that uses LLMs to analyze resumes, match them against job descriptions, and track applications รข built with React, TypeScript, and AWS serverless. I built it to solve my own job search problem, so I was the first real user."

### รจรฅรฅยฉรฅรฅยฐรฆยกรงรฉรคยฟ
| รฅยฐรฆยก | รฅยฑรงยคยบรฉรฉยป |
|------|---------|
| Aussie EcoLens | AWS / GCPรฃDockerรฃImage ClassificationรฃCloud Architectureรฏยผรฅรฉรฅยฐรฆยกรฏยผ|
| Lumoraรฏยผรฆยซรฅรฏยผ| React รฅรงยซรฃCanvas APIรฃรจยฆรจยฆยบรจยจยญรจยจรฅยทรจยกรฅ |
| **Career Navigator** | **LLM IntegrationรฃPrompt EngineeringรฃDashboard รจยจยญรจยจรฃProduct ThinkingรฃRecommendation System** |

> รฐยก รคยธรฅรฅยฐรฆยกรฆยพรฅยจรคยธรจยตยทรฏยผรฅรฅยฅยฝรจยฆรจรฃCloud / รจยฆรจยฆยบรฅยทรจยก / AI รงยขรฅรฆรงยถยญรฃรคยธรฅรคยธรฅรงรจยฝรฅรฉยขรฏยผรฅยฑยฅรฆยญยทรคยธรคยธรฆรงรจยตยทรคยพรฉรจยครฃ

## 2. รงยบรคยปรฉยบยผรฅรฉรฅรฅยฐรฆยก Positioning

### รฅยพ Lumora รจยฝรฉรคยพรงรงรงยฑ
Lumora รงรฅยฅรฅยนยปรฆยถรฉรจยฆรจยฆยบรฉรฅยฟรฏยผรฅยถรฅยฏยฆรฆยดรฉยฉรฅรงยจ Godot รคยนรฉยกรงรฉรฆยฒรฅยผรฆรคยพรฅรฏยผรงยจ React + Canvas รงยกยฌรฅยปรฅรงยซรฏยผรฆรฅยฅรงยขรฅยบรฆยฏรคยธรฅรงยฎรฃCareer Navigator รฅรฅรฅยฅยฝรงยธรฅรขรขรฅยฎรงรฆ ยธรฅยฟรฅยนรฅยผรฆยฏ dashboardรฃรจยณรฆรฆยตรฃAI รฆยดรฅรฏยผรฉรฆยญยฃรฅยฅยฝรฆยฏ React + TypeScript รฆรฆรฉยทรงรฉ รฅรฏยผรคยนรฆยดรงยดรฆยฅรฅยฐรฆรคยฝ รงยฎรฅรฅยจรฆรง Frontend / Full Stack รจยทรงยผยบรฃ

### รคยฝยฟรงยจรจรงรฉยป
รฆยฑรจยทรจรงรจยณรจยจรฉรฅยธยธรฅรฆยฃรฅยจ Resume / LinkedIn / Seek / Indeed / Excel / Notion / ChatGPT รคยนรฉรฏยผรฅยฐรจยดรฏยผ
- รคยธรงยฅรฉรจยชรฅยทยฑรงยผยบรคยปรฉยบยผรฆรจยฝ
- รคยธรงยฅรฉรฅยฑยฅรฆยญยทรฆยฏรฅยฆรงยฌยฆรฅรจยทรงยผยบ
- รคยธรงยฅรฉรฆรคยบรฅยชรคยบรฅยฌรฅยธรฃรฉยฒรฅยบยฆรฅยฐรฅยช
- รคยธรงยฅรฉรคยธรคยธรฆยญยฅรจยฉยฒรฅยญยธรคยปรฉยบยผ

### รจยงยฃรฆยฑยบรฆยนรฆยก
รฆ **Career Goal + Resume + Job Applications + AI Analysis** รฆยดรฅรฅยฐรฅรคยธรฅรฅยนยณรฅยฐรฏยผรฅยฝยขรฆรคยธรฅรฅยฎรฆยดรคยฝรฅยฐรฅยทยงรงรจยทรฆยถยฏรงยฎยกรงรจยฟยดรฅรฃ

### รฅยทยฎรงยฐรฅรฆรคยบรฏยผรฉยขรจยฉยฆรงยจรฏยผ
> "Aussie EcoLens รฅยฑรงยคยบรคยบรฆรฅยจรฅรฉรงยฐรฅยขรคยธรฅ cloud architecture รงรจยฝรฅรฏยผCareer Navigator รฅยฑรงยคยบรคยบรฆรงยจรงยซรจยจยญรจยจรคยธรฅ AI รงยขรฅรงรจยฝรฅรขรขรฅยพรคยฝยฟรงยจรจรงรฉยปรฅยบรงยผรฃรฅ prompt designรฃรจรงรงรฅยฏยฆรคยธรงรงรฉรงยตรฆยงรฅรจยณรฆรฏยผPDF รฅยฑยฅรฆยญยทรฏยผรฃ"

## 6. รฉรฅยฐรฅรงยดรฉรจยผยฏ Match Level Logic

> รข รฏยธ **V2 รฉรฅยคยงรจยชยฟรฆยดรฏยผรฆรฅรฆยฌรงรฃรฉรฅยฐรฅรฆยธ 0รข100%รฃรฆยนรฆรคยบรงยดรฅรฉยกรฃ**
> รงยฒยพรงยขยบรงยพรฅรฆยฏรงรจยตยทรคยพรงยฒยพรฆยบรฏยผรคยฝรฅยถรฅยฏยฆรงยฎรคยธรฅยบรฉยฃรฅรงยฒยพรงยขยบรฅยบยฆรฏยผ82 รจยท 79 รฆยฒรฆรงรฅยฏยฆรฅยทยฎรฅยฅรฏยผรฃรฉยขรจยฉยฆรฅยฎรคยธรฅรฃรงยบรคยปรฉยบยผรฆยฏ 82 รคยธรฆยฏ 80รฃรฅยฐยฑรงยญรคยธรฅยบรคยพรฏยผรฅรจรฆยฃรฅรฃรคยบรงยดรฅรฉยกรฆยดรจยช รฅยฏยฆรฃรฅยฐรคยฝยฟรงยจรจรฆยดรฆรงยจรฃLLM รจยผยธรฅยบรคยนรฆยดรงยฉยฉรฅยฎรขรขรฉรจยท Skill Tree รงยจรฅรฉยกรฅรคยปยฃรงรฆรฆยฏรฅรคยธรฅรฉรจยผยฏรฃ

### รคยบรฅรงยญรงยด
```typescript
type MatchLevel =
  | "strong"      // รฅยผยทรฅยนรฉรฏยผJD รง required รฆรจยฝรฅยคยงรฅยครฅยฝรคยธยญ
  | "good"        // รจยฏรฅยฅยฝรฏยผรฅยฝรคยธยญรฅยครฆยธรฏยผรงยผยบ 1รข2 รฅรฆยฌยกรจยฆรฆรจยฝ
  | "moderate"    // รคยธยญรงยญรฏยผรฅยฝรคยธยญรงยดรคยธรฅรฏยผรงยผยบรฅยนยพรฅรฉรจยฆรฆรจยฝ
  | "weak"        // รฅรฅยผยฑรฏยผรงยผยบรฅยครฆยธรฆ ยธรฅยฟรฆรจยฝ
  | "mismatch";   // รคยธรฅยนรฉรฏยผรฆยนรฅรคยธรงยฌยฆ

type JobMatchResult = {
  matchLevel: MatchLevel;
  matchedSkills: string[];
  missingSkills: string[];
  suggestedKeywords: string[];
};
```

### รฅรงยดรจรฅยพรงรฅยฏรจยงยฃรฉรจยฆรฅ
รฉรงยถรฉยกยฏรงยคยบรงยตยฆรคยฝยฟรงยจรจรงรฆยฏรงยญรงยดรฏยผรคยฝรฅยบรฅยฑยครฆรคยธรฅรจยฝรจยฌรฆยธรฆยฅรงรจยจรงยฎรฉรจยผยฏรฏยผรฉยขรจยฉยฆรจยขยซรฅรฆรฅยฏรคยปยฅรจยงยฃรฉรฏยผรฏยผ

```
1. LLM รฅยพ JD รฆยฝรฅยบรฆรจยฝรฆยธรฅยฎรฏยผรคยธยฆรฆยจรจยจรฆยฏรฅรฆยฏ required รฉรฆยฏ preferred
2. รจรฃรฅยทยฒรฅยญรงรฅยฑยฅรฆยญยทรฅรฆรฆรจยฝรฃรฆยฏรฅยฐรฏยผรงยฎรฅยบ required รฅยฝรคยธยญรง
3. รคยพรฅยฝรคยธยญรงรฅยฐรฆรฅยฐรงยญรงยดรฏยผ
   - required รฅยฝรคยธยญ รขยฅ 80%        รข strong
   - required รฅยฝรคยธยญ 60รข79%       รข good
   - required รฅยฝรคยธยญ 40รข59%       รข moderate
   - required รฅยฝรคยธยญ 20รข39%       รข weak
   - required รฅยฝรคยธยญ < 20%        รข mismatch
```

> รฐยก รฉรฉยปรคยธรฆยฏรฉรคยบรฆยธรฅยญรฉรฆยชยปรฅยครงยฒยพรฆยบรฏยผรจรฆยฏ**รฆยดรฅยฅรจยฆรฅรฆยฏรฅยฏรจยงยฃรฉรง**รฃรฉยขรจยฉยฆรฆรคยฝ รจยฝรจยชยชรฃรฆรงยจ required รฆรจยฝรฅยฝรคยธยญรงรฅรงยดรฏยผรจรคยธรฆยฏรจยฎ LLM รฉยจรคยพยฟรงยตยฆรคยธรฅรฅรฆยธรฃรขรขรฉรฅยฑรงยคยบรงรฆยฏรงยขรฅรฅยครฆยทรฅรฏยผรคยธรฆยฏรฅรงยฒยพรงยขยบรฃ

### รงยบรคยปรฉยบยผรคยธรงยจรงยพรฅรฆยฏรฏยผรฉยขรจยฉยฆรจยชยชรฆยณรฏยผ
> "I deliberately avoided showing a precise match percentage. A number like 82% looks rigorous but isn't actually meaningful รข there's no real difference between 82 and 79. I used five interpretable levels backed by a required-skill hit-rate rule. It's more honest, and more useful: a job seeker wants to know 'should I apply', and 'good' answers that better than '78%'."

### Prompt รจยจยญรจยจรฅรฅรฏยผMVPรฏยผ
- รฆยฐยธรฉ รจยฆรฆยฑ LLM รฅรฅยณ **รงยด JSON**รฏยผรคยธรจยฆรฅยคยพรฉรจยชยชรฆรฆรฅยญรฏยผรฅรงยซยฏรฆรจยฝรงยฉยฉรฅยฎ parse
- รฅยฑยฅรฆยญยทรฅรฆรจ JD รฉรฅยฐรฉยฝรงยจ **รฅยบรฅยฎ schema**รฏยผ
```typescript
type ResumeAnalysis = {
  extractedSkills: string[];
  projects: string[];
  experience: string[];
  suggestions: string[];
};

type JobMatchResult = {
  matchLevel: MatchLevel;        // รจยฆรงยฌยฌ 6 รงยซ 
  matchedSkills: string[];
  missingSkills: string[];
  suggestedKeywords: string[];
};
```
- LLM รจยงยฃรฆรฅยคยฑรฆรฏยผรฆ ยผรฅยผรจยทรฆรฏยผรฆรฏยผรฅรงยซยฏรจยฆรฆ fallback UIรฏยผรคยธรจยฆรฆยดรฉ รฅยดรฉยฏรจยชยครจยจรฆยฏรฏยผรจยฆรงยฌยฌ 10 รงยซ รฏยผ

**รฅรฅ รฏยผ** MVP รจยฆรงรฆยฏรงยฉยฉรฅยฎรฅยฏรฉ รฆยธยฌรงรจยผยธรฅยบรฏยผรฅรงยจรงยฐยกรฅยฎรฅยผรฅยซรฉยฉรจยญรฅยนรฅยผรฏยผรฉยฟรฅรคยธรฉรฅยงรฅยฐยฑรฅ RAG รฅยขรฅ รจยครฉรฅยบยฆรฃ

## 17. รฉยขรจยฉยฆรจยชยชรฆยณ Interview Narrative

### รคยธรฅยฅรจยฉยฑรคยปรงยดยน
> "I built an AI-powered career navigation platform that parses resumes, matches them against job descriptions, and tracks applications. I used it myself during my own job search รข so I was the first real user."

### รฆ ยธรฅยฟรฆรจยกรฆยฑยบรงยญรจยชยชรฆยณ

**รงยบรคยปรฉยบยผ MVP รงยฏรฅรง รฉรฉยบยผรฅยฐรฏยผ**
> "I originally scoped this with way too many features at once. I cut it down to the smallest loop that proves value รข upload resume, get analysis, match against a job description, track the application รข and pushed everything else to later phases."

**รฉรฅยฐรงยบรคยปรฉยบยผรงยจรงยญรงยดรจรคยธรฆยฏรงยพรฅรฆยฏรฏยผ**
> "A number like 82% looks rigorous but isn't meaningful รข there's no real difference between 82 and 79. I used five interpretable levels backed by a required-skill hit-rate rule. It's more honest and more useful: a job seeker wants to know 'should I apply', and 'good' answers that better than a fake percentage."

**Skill Tree รงยบรคยปรฉยบยผรคยธรฆยฏรฅยณรฆ AI รงรฆรฏยผ**
> "Fully dynamic generation looked impressive on paper but was unreliable for a live demo. I built curated skill trees for common roles and let the AI handle classification instead of generation รข same lesson I learned on Lumora, where I replaced raw AI output with a stable mapping layer."

**รงยบรคยปรฉยบยผรฅยฑยฅรฆยญยทรฅยชรฅรฆรคยธรฆยฌยกรฏยผ**
> "Users compare one resume against many jobs. Re-running the LLM analysis every time would waste calls and time, so I analyze once on upload, store the structured result, and reuse it for every job match."

**รงยบรคยปรฉยบยผรฅยฑยฅรฆยญยทรจยฆรฉยทรฆรคยฟรฅยญรฏยผรจยทรคยฝ รฅยฆรคยธรฅรฅยฐรฆยกรงรฃรงยงรงรงยจรฅยฎรฅยฐยฑรฅยชรฃรฉรจยผยฏรคยธรคยธรฆยจยฃรฏยผ**
> "Different asset model. In Lumora the photo was just a trigger รข no lasting value after recognition. Here the resume *is* the core asset users come back to, so it has to persist. That also meant handling PII and encryption, which Lumora didn't need."
